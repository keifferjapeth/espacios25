# espacios25

Static site for the Espacios25 simulation demo.

## Run locally

1. Clone or download this repository.
2. Open `index.html` in your browser (double-click the file or drag it into the browser window).
3. For a simple local server, run:
   ```bash
   python -m http.server 8000
   ```
   then visit http://localhost:8000/ in your browser.
